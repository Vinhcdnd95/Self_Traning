1. Cài đặt công cụ
Tổng quan 
NestJS là một framework Node.JS cho phép xây dựng ứng dụng phía server. Nest mở rộng các framework Node.js như Express 
hay Fastify để bổ sung thêm nhiều module hay thư viện hỗ trợ việc xử lý tác vụ. 
Đây là một framework mã nguồn mở, sử dụng TypeScript và rất linh hoạt để xây dựng các hệ thống backend.
2. Kiến trúc chung của NestJs
Dist: dịch code sang js
Node_modules: cài đặt thư viện
Src: mã nguồn dự án
Tsconfig.json: file cấu hình dự án
Tsconfig.build.json: quá trình config
Eslintr.js: cách code typrscript
